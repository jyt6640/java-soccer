# 축구 경기 결과 분석기

두 팀의 이름과 점수를 입력받아 승패를 판단하는 애플리케이션입니다.

## 기능

- 두 팀의 이름과 점수 입력
- 점수 비교를 통한 승패 판정
- 무승부 처리
- 입력값 유효성 검증
  - 팀 이름: 최대 10자
  - 점수: 0-99 범위

## 상세 기능 목록

### 1. 입력 기능
- [x] 게임 진행을 위한 입력을 받는다.
    - **팀 이름**과 **해당 팀의 점수**를 입력받는다. (예: `팀A 3`)
- [x] 입력값은 지정된 구분자를 기준으로 파싱한다.

### 2. 게임 로직 (핵심 도메인)
- [x] **점수 비교 및 승패 판단**
    - 두 팀의 점수를 비교하여 승리, 무승부, 패배를 판단한다.
    - 예: A팀 3점, B팀 2점 -> A팀 승리
- [x] **경기 기록 생성**
    - 각 경기의 결과(팀 이름, 점수, 승패)를 생성하고 관리한다.

### 3. 출력 기능
- [x] 경기 결과를 사용자에게 출력한다.
    - 승리한 팀, 혹은 무승부 결과를 포맷에 맞춰 출력한다.
- [x] 예외 발생 시 에러 문구(`[ERROR]`)를 출력한다.

### 4. 예외 처리 (Validation)
- [x] **팀 이름 유효성 검사**
    - 팀 이름이 비어있거나 공백인 경우
- [x] **점수 유효성 검사**
    - 점수가 숫자가 아닌 경우
    - 점수가 음수인 경우
- [x] **입력 형식 검사**
    - 입력 형식이 지정된 포맷과 다른 경우

## 사용 방법

```
축구 경기 결과 분석기 입니다.
팀 이름과 점수를 입력해주세요. (예시: 팀A 3, 팀B 2)

팀A 3
팀B 2

팀A 승리
```

### 입력 형식
- 형식: `{팀이름} {점수}`
- 예시: `팀A 3`, `Korea 5`

### 출력 형식
- 승리: `{팀이름} 승리`
- 무승부: `무승부`

### 에러 메시지
| 상황 | 메시지 |
|------|--------|
| 잘못된 입력 형식 | `[ERROR] 입력 형식이 올바르지 않습니다.` |
| 음수 점수 | `[ERROR] 점수는 음수 일 수 없습니다.` |
| 100점 이상 | `[ERROR] 점수는 100점 이상 일 수 없습니다.` |
| 팀 이름 10자 초과 | `[ERROR] 팀 이름은 10자 미만으로 입력해주세요.` |

## 프로젝트 구조

```
src/main/java/soccer/
├── Application.java              # 진입점
├── controller/
│   └── SoccerController.java     # 애플리케이션 흐름 제어
├── service/
│   ├── MatchService.java         # 승패 판정 비즈니스 로직
│   └── dto/
│       └── WinnerName.java       # 결과 DTO
├── domain/
│   ├── Team.java                 # 팀 엔티티 (이름 검증)
│   ├── Score.java                # 점수 엔티티 (범위 검증)
│   └── TeamStatus.java           # 팀+점수 상태 객체
├── view/
│   ├── InputView.java            # 사용자 입력 처리
│   └── OutputView.java           # 결과 출력 처리
├── util/
│   ├── InputParser.java          # 입력 파싱
│   └── MatchInput.java           # 파싱 결과 DTO
└── common/constant/
    └── ErrorMessage.java         # 에러 메시지 상수
```

## 아키텍처

### 계층 구조
```
Controller → Service → Domain
     ↓
   View
```

### 주요 컴포넌트

**Controller**
- `SoccerController`: 사용자 입력을 받아 서비스 호출 후 결과 출력

**Service**
- `MatchService`: 두 팀의 점수를 비교하여 승자 결정

**Domain**
- `Team`: 팀 이름 (최대 10자 검증)
- `Score`: 점수 (0-99 범위 검증, `isHigherThan()` 메서드)
- `TeamStatus`: Team + Score 조합

**View**
- `InputView`: 콘솔 입력
- `OutputView`: 결과/에러 출력
